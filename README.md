# BDC Tooltip for Arena Breakout Infinite

This is a simple UserScript to enhance the "BDC" (Ban, Discipline, and Complaint) system for the game Arena Breakout Infinite.

## Features

*   **Video Speed Controls:** Adds a dropdown menu to the video player to control the playback speed (from 0.25x to 2.0x).
*   **English Translations:** Changes the placeholder text for the violation description textareas to English.
*   **No Watermark:** Hides the watermark that appears over the video.

## Installation

1.  **Install a UserScript Manager:** You need a browser extension to run this script. Popular choices include:
    *   [Tampermonkey](https://www.tampermonkey.net/) (for Chrome, Firefox, Edge, Safari)
    *   [Greasemonkey](https://www.greasespot.net/) (for Firefox)
    *   [Violentmonkey](https://violentmonkey.github.io/) (for Chrome, Firefox, Edge)

2.  **Install the Script:**
    *   Go to the `content.js` file in this repository.
    *   Click the "Raw" button to view the raw code.
    *   Your UserScript manager should automatically detect the script and ask you to install it. Confirm the installation.

3.  **Usage:**
    *   Navigate to the Arena Breakout Infinite BDC page. The script will automatically run and apply the enhancements.
